Поиск общих касательных к двум окружностям Search for common tangents to two circles

Даны две окружности. Two circles are given. Требуется найти все их общие касательные, т.е. It is required to find all their common tangents, i.e. все такие прямые, которые касаются обеих окружностей одновременно. All such lines that touch both circles simultaneously.

Описанный алгоритм будет работать также в случае, когда одна (или обе) окружности вырождаются в точки. The described algorithm will work also in the case when one (or both) circles degenerate into points. Таким образом, этот алгоритм можно использовать также для нахождения касательных к окружности, проходящих через заданную точку. Thus, this algorithm can also be used to find tangents to a circle passing through a given point.

Количество общих касательных Number of common tangents

Сразу отметим, что мы не рассматриваем вырожденные случаи: когда окружности совпадают (в этом случае у них бесконечно много общих касательных), или одна окружность лежит внутри другой (в этом случае у них нет общих касательных, или, если окружности касаются, есть одна общая касательная). We note at once that we do not consider degenerate cases: when the circles coincide (in this case they have infinitely many common tangents), or one circle lies inside the other (in this case they do not have common tangents, or if the circles touch, there is one common tangent).

В большинстве случаев, две окружности имеют четыре общих касательных. In most cases, two circles have four common tangents.

Если окружности касаются , то у них будет три обших касательных, но это можно понимать как вырожденный случай: так, как будто две касательные совпали. If the circles touch , then they will have three common tangents, but this can be understood as a degenerate case: as if two tangents coincided.

Более того, описанный ниже алгоритм будет работать и в случае, когда одна или обе окружности имеют нулевой радиус: в этом случае будет, соответственно, две или одна общая касательная. Moreover, the algorithm described below will work also in the case when one or both circles have a zero radius: in this case there will be, respectively, two or one common tangent.

Подводя итог, мы, за исключением описанных в начале случаев, всегда будем искать четыре касательные . Summing up, we, except for the cases described at the beginning, will always look for four tangents . В вырожденных случаях некоторые из них будут совпадать, однако тем не менее эти случаи также будут вписываться в общую картину. In degenerate cases, some of them will coincide, but nevertheless these cases will also fit into the overall picture.

Реализация Implementation

Опишем сначала все необходимые структуры данных и другие вспомогательные определения: We first describe all the necessary data structures and other auxiliary definitions: 

 struct pt {
	double x, y ;
 
	pt operator - ( pt p ) {
		pt res = { x - p. x , y - p. y } ;
		return res ;
	}
} ;
 
struct circle : pt {
	double r ;
} ;
 
struct line {
	double a, b, c ;
} ;
 
const double EPS = 1E-9 ;
 
double sqr ( double a ) {
	return a * a ;
}

Тогда само решение можно записать таким образом (где основная функция для вызова — вторая; а первая функция — вспомогательная): Then the solution itself can be written in this way (where the main function for the call is the second one, and the first function is the auxiliary one): 

 void tangents ( pt c, double r1, double r2, vector < line > & ans ) {
	double r = r2 - r1 ;
	double z = sqr ( c. x ) + sqr ( c. y ) ;
	double d = z - sqr ( r ) ;
	if ( d < - EPS )  return ;
	d = sqrt ( abs ( d ) ) ;
	line l ;
	l. a = ( c. x * r + c. y * d ) / z ;
	l. b = ( c. y * r - c. x * d ) / z ;
	l. c = r1 ;
	ans. push_back ( l ) ;
}
