    Linear Congruence Equation

    Table of Contents

    Solution with the Advanced Euclidean Algorithm

    This equation is of the form:

    a⋅x=b(modn),a⋅x=b(modn),

    where aa, bb and nn are given integers and xx is an unknown integer.

    It is required to find the value xx from the interval [0,n−1][0,n−1] (clearly, on the entire number line there can be infinitely many solutions that will differ from each other in n⋅kn⋅k , where kk is any integer). If the solution is not unique, then we will consider how to get all the solutions.

    Solution by finding the inverse element

    Let us first consider a simpler case where aa and nn are coprime (gcd(a,n)=1gcd(a,n)=1). Then one can find the inverse of aa, and multiplying both sides of the equation with the inverse, and we can get a unique solution.

    x=b⋅a−1(modn)x=b⋅a−1(modn)

    Now consider the case where aa and nn are not coprime (gcd(a,n)!=1gcd(a,n)!=1). Then the solution will not always exist (for example 2⋅x=1(mod4)2⋅x=1(mod4) has no solution).

    Let g=gcd(a,n)g=gcd(a,n), i.e. the greatest common divisor of aa and nn (which in this case is greater than one).

    Then, if bb is not divisible by gg, there is no solution. In fact, for any xx the left side of the equation a⋅x(modn)a⋅x(modn) , is always divisible by gg, while the right-hand side is not divisible by it, hence it follows that there are no solutions.

    If gg divides bb, then by dividing both sides of the equation by gg (i.e. dividing aa, bb and nn by gg), we receive a new equation:

    a′⋅x=b′(modn′)a′⋅x=b′(modn′)

    in which a′a′ and n′n′ are already relatively prime, and we have already learned how to handle such an equation. We get as solution for x′x′.

    It is clear that this x′x′ will also be a solution of the original equation. However it will not be the only solution. It can be shown that the original equation has exactly gg solutions, and they will look like this:

    xi=(x′+i⋅n′)(modn)for i=0…g−1xi=(x′+i⋅n′)(modn)for i=0…g−1

    Summarizing, we can say that the number of solutions of the linear congruence equation is equal to either g=gcd(a,n)g=gcd(a,n) or to zero.

    Solution with the Advanced Euclidean Algorithm

    We can rewrite the linear congruence to the following Diophantine equation:

    a⋅x+n⋅k=b,a⋅x+n⋅k=b,

    where xx and kk are unknown integers.

    The method of solving this equation is described in the corresponding article Linear Diophantine equations and it consists of applying the Extended Euclidean Algorithm.

    It also describes the method of obtaining all solutions of this equation from one found solution, and incidentally this method, when carefully considered, is absolutely equivalent to the method described in the previous section.
