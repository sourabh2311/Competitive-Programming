    Finding Power of Factorial Divisor

    Table of Contents
    Prime kk

    Composite kk

    You are given two numbers nn and kk. Find the largest power of kk xx such that n!n! is divisible by kxkx.

    Prime kk

    Let's first consider the case of prime kk. The explicit expression for factorial

    n!=1⋅2⋅3…(n−1)⋅nn!=1⋅2⋅3…(n−1)⋅n

    Note that every kk-th element of the product is divisible by kk, i.e. adds +1+1 to the answer; the number of such elements is ⌊nk⌋⌊nk⌋.

    Next, every k2k2-th element is divisible by k2k2, i.e. adds another +1+1 to the answer (the first power of kk has already been counted in the previous paragraph). The number of such elements is ⌊nk2⌋⌊nk2⌋.

    And so on, for every ii each kiki-th element adds another +1+1 to the answer, and there are ⌊nki⌋⌊nki⌋ such elements.

    The final answer is

    ⌊nk⌋+⌊nk2⌋+…+⌊nki⌋+…⌊nk⌋+⌊nk2⌋+…+⌊nki⌋+…

    The sum is of course finite, since only approximately the first logknlogk⁡n elements are not zeros. Thus, the runtime of this algorithm is O(logkn)O(logk⁡n).

    Implementation


    int fact_pow (int n, int k) {
    int res = 0;
    while (n) {
        n /= k;
        res += n;
    }
    return res;
}

    Composite kk

    The same idea can't be applied directly. Instead we can factor kk, representing it as k=kp11⋅…⋅kpmmk=k1p1⋅…⋅kmpm. For each kiki, we find the number of times it is present in n!n! using the algorithm described above - let's call this value aiai. The answer for composite kk will be

    mini=1…maipimini=1…maipi
