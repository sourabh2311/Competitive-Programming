      We can reduce the time complexity to approx. (2^(n/2))log(n/2) if no. of integers >s/2 are more than n/2 just by
      1. Sorting the whole array (assume in increasing) ,then
      2. Divide the array in two halves , such that first array contains element <=s/2 and rest in 2nd array
      3. Now , make all possible subsets of 1st array , taking subsets in count whose sum is <=s , we have to choose atmost one element if possible from 2nd array (as 2nd array contains element >s/2) by binary search taking approx. log(n/2) time .
